Bedrijfsnetwerk Setup met Cisco Packet Tracer

Datum:
November 2024
Categorie:
Networking & Infrastructure
Status:
Voltooid

Over dit project

Voor dit project heb ik een compleet bedrijfsnetwerk ontworpen en geïmplementeerd in Cisco Packet Tracer. Het netwerk ondersteunt een kantoorpand van twee verdiepingen met werkstations, printers, VoIP telefoons en draadloze toegang. De opdracht vereiste een professionele rack-opstelling met redundantie en proper cable management.

Het ontwerp omvat gesegmenteerde VLANs voor verschillende afdelingen, een dedicated server rack met file server en backup server, en een robuuste internetverbinding via een router met failover mogelijkheden.

Netwerkarchitectuur

Het netwerk is opgebouwd rond een hiërarchisch model met de volgende componenten:

  • Core Layer: 48-poorts Gigabit switch (TP-Link TL-SG1048) als centrale verdeler
  • Access Layer: 24-poorts switch (TP-Link TL-SG1024) voor eerste verdieping
  • Router: ASUS RT-AX53U voor internet en inter-VLAN routing
  • Wireless: 3x Mikrotik wAP ax WiFi6 access points
  • Servers: File server en backup server in 12U rack

Gebruikte Technologieën

  • Cisco Packet Tracer
  • VLANs
  • Subnetting
  • STP
  • DHCP
  • VoIP
  • WiFi6
  • Cat6A

Rack Layout

De 12U rack is als volgt ingedeeld:

Positie Component Beschrijving
1U Cable Management Panel Kabelgeleiding boven patchpanelen
2U Patchpanel 1 Aansluitingen gelijkvloers
3U Patchpanel 2 Aansluitingen eerste verdieping
4U Patchpanel 3 Printers + Access Points
5U Cable Management Panel Kabelgeleiding onder patchpanelen
6U Core Switch (48-poorts) Hoofdswitch voor alle bekabelde verbindingen
7U Access Switch (24-poorts) Switch voor uplink naar verdieping
8-9U File Server Rackmount 2U server
10-11U Backup Server Rackmount 2U server
12U Router Verbindt netwerk met internetprovider

Uitdagingen & Oplossingen

VLAN Segmentatie: Het scheiden van verschillende afdelingen en device types vereiste zorgvuldige planning. Ik heb elk departement een eigen VLAN toegewezen met inter-VLAN routing via de router voor gecontroleerde communicatie tussen segmenten.

VoIP Implementatie: Het integreren van 24 VoIP telefoons vereiste een dedicated voice VLAN met QoS instellingen om voice traffic prioriteit te geven boven data traffic.

Redundantie: Door Spanning Tree Protocol (STP) te configureren op de switches wordt loop prevention gegarandeerd, terwijl de backup server automatische failover biedt voor de file server.

Schaalbaarheid: Het netwerk is ontworpen met toekomstige uitbreiding in gedachten. De 48-poorts core switch heeft voldoende vrije poorten en het IP schema biedt ruimte voor groei.

  • 24 Desktop PC's: €8.376
  • 24 VoIP Telefoons: €3.000
  • Servers (File + Backup): €3.022,40
  • Netwerk apparatuur: €1.216,25
  • Bekabeling en rack: €1.120

Resultaat

Het netwerk functioneert stabiel en ondersteunt alle vereiste diensten. Alle 24 werkstations hebben betrouwbare bekabelde connectiviteit, de VoIP telefoons werken zonder packet loss, en de wireless access points bieden volledige dekking over beide verdiepingen.

Het project demonstreert praktische kennis van netwerk design principes, VLAN segmentatie, proper cabling standards en enterprise-grade infrastructuur planning.
