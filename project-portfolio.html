Over dit project

Deze portfolio website is gebouwd om mijn werk en vaardigheden te presenteren aan potentiële werkgevers en opdrachtgevers. Het project was een uitstekende oefening in modern web development met focus op responsive design, semantic HTML en accessibility.

De site is volledig responsive en werkt naadloos op alle devices, van smartphones tot desktop computers. Door gebruik te maken van Bootstrap 5 kon ik snel een professioneel design realiseren zonder veel custom CSS te schrijven.

Design Principes

Bij het ontwerp van deze site heb ik me gefocust op de volgende principes:

  • Semantic HTML: Gebruik van correcte HTML5 elementen (article, section, nav, aside) voor betere SEO en accessibility
  • Mobile-First: Ontwerp begint bij kleine schermen en schaalt op naar grotere displays
  • Performance: Minimale JavaScript, optimized images, en gebruik van CDN voor frameworks
  • Accessibility: Proper heading hierarchy, alt teksten, ARIA labels waar nodig
  • Clean Code: Geen onnodige divs, maximaal gebruik van Bootstrap utilities

Structuur

De website bestaat uit vijf hoofdpagina's:

  • Home: Introductie met hero section, uitgelichte projecten, vaardigheden en call-to-action
  • Portfolio: Overzicht van alle projecten in een responsive grid layout
  • Project Details: Individuele pagina's per project met uitgebreide informatie
  • Contact: Contactformulier (Formspree integration) en contactgegevens

Technische Implementatie

Responsive Grid: Bootstrap's grid systeem zorgt voor perfecte layouts op alle schermformaten. Project cards gebruiken col-12 col-sm-6 col-md-4 voor 1, 2 of 3 kolommen afhankelijk van viewport width.

Formulier Integratie: Het contact formulier gebruikt Formspree voor server-side processing zonder eigen backend nodig te hebben. Dit houdt de site statisch en snel.

Navigation: Mobile hamburger menu met Bootstrap's collapse component, automatisch actieve state highlighting op basis van huidige pagina.

Custom Styling: Minimale custom CSS beperkt tot color scheme, custom button styles, en enkele layout tweaks. Alles wat Bootstrap kan, wordt via Bootstrap gedaan.

Color Scheme

De gekozen kleuren zijn gebaseerd op een oranje-blauw palette:

  • Primary: #F24F13 - Gebruikt voor headings, buttons en accenten
  • Primary Light: #F26430 - Hover states
  • Text Dark: #113743 - Body tekst
  • Text Muted: #587e8c - Secondary tekst

Uitdagingen & Oplossingen

Semantic HTML vs Divitis: In het begin gebruikte ik te veel divs. Door bewust te kiezen voor article, section, nav en aside werd de HTML cleaner en betekenisvoller.

Bootstrap Overrides: Ik wilde custom kleuren maar Bootstrap's default stijlen overschrijven was tricky. Opgelost door CSS custom properties te gebruiken en specifieke classes zoals .btn-primary-custom.

Form Handling: Zonder backend was form processing een uitdaging. Formspree bood een elegante oplossing zonder complexiteit.

Image Placeholders: Nog geen echte project screenshots. Momenteel gebruiken project cards placeholder images, deze worden vervangen zodra projecten live gaan.

Best Practices

Tijdens dit project heb ik enkele belangrijke web development best practices toegepast:

  • Gebruik van semantic HTML5 elementen i.p.v. generieke divs
  • Proper heading hierarchy (h1 → h2 → h3) voor accessibility
  • Alt teksten op alle images
  • ARIA labels op social media links
  • rel="noopener" op externe links voor security
  • Consistent gebruik van Bootstrap utilities i.p.v. custom CSS
  • Mobile-first responsive design
  • Fast loading door gebruik van CDN en minimal custom code

Resultaat

De website is live en volledig functioneel. Het design is clean, professional en laat mijn projecten goed tot hun recht komen. De site scoort uitstekend op Google Lighthouse met hoge scores voor Performance, Accessibility en SEO.
